# Variables

Variables are used in .less files to standardize css values. They are represented by css variable. This is an example how to use
Upfluence css variables:

`
.your-css-class {
css-attribute: var(--custom-vaiables);
}
`

## Box shadow

The following values represent box shadow level. There is 3 types of levels: sm, md, lg.

| Name            | Value                       |
|-----------------|-----------------------------|
| --box-shadow-sm | 0 1px 3px 0 #1B1E2110       |
| --box-shadow-md | 0 10px 15px -3px #1B1E2110  |
| --box-shadow-lg | 0 25px 50px -10px #1B1E2120 |

## Backdrop color

Color backdrop is the color for the modal backdrop.

| Name             | Value     |
|------------------|-----------|
| --color-backdrop | #1B1E2160 |


## Border color

The border default color is used in all borders in component. The content separator color is used to separate different elements, for example
to separate lines of a table.

| Name                      | Value                 |
|---------------------------|-----------------------|
| --color-border-default    | var(--color-gray-200) |
| --color-content-separator | var(--color-gray-100) |


## Border radius

The following values represent border radius level. There is 2 types of levels: sm, md.

| Name               | Value |
|--------------------|-------|
| --border-radius-sm | 4px   |
| --border-radius-md | 8px   |
