# Colors

New color palette use CSS variables.

Note: The most important variables can be used as html classes, calling them as `background-color-[color]-[weight]`.

E.G. `<div class="background-color-primary-50 font-color-gray-900">`

## Primary

Primary and secondary color.

<div class="fx-col fx-gap-px-20">
  <div class="fx-row">
    <div class="palette-container">
      {
        [
          {variable: 'color-primary-50', hex: '#ECECFD'},
          {variable: 'color-primary-400', hex: '#3232F6'},
          {variable: 'color-primary-500', hex: '#0D0DE6'},
          {variable: 'color-primary-600', hex: '#0404C7'},
          {variable: 'color-primary-900', hex: '#060666'}
        ].map((token, i) => {
          return(
            <div key={i} class={token.variable + " color-container-v2"}>
              <span>--{token.variable}</span>
              <span>{token.hex}</span>
            </div>
          )
        })
      }
    </div>
  </div>
</div>
