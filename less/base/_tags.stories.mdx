# Tags

Tags are used to display informations to the user like state or status. Built around the `upf-tag` class, tags have a
total for six different types:
- regular with `upf-tag--regular` class
- success with `upf-tag--success` class
- alert with `upf-tag--alert` class
- critical with `upf-tag--critical` class
- neutral with `upf-tag--neutral` class
- inactive with `upf-tag--inactive` class

Tags take into account the possibility of adding an icon inside
<Canvas>
  <div class="flex-row tags-container__demo">
    <div class="upf-tag upf-tag--alert margin-right-sm">
      <i class="fa fa-star"></i>
      Favorite
    </div>
    <div class="upf-tag upf-tag--success">
      <i class="fa fa-envelope-open"></i>
      Opened
    </div>
  </div>
</Canvas>

It's possible to limit the size of the text inside the tag. To do that, the text content must be put in `span`. A text ellipsis
is applied and the `max-width` is limit to 80px.

<Canvas>
  <div class="flex-row">
    <div class="upf-tag upf-tag--neutral margin-right-x-sm">
      <span>Short</span>
    </div>
    <div class="upf-tag upf-tag--neutral margin-right-x-sm">
      <span>Medium tag</span>
    </div>
    <div class="upf-tag upf-tag--neutral">
      <span>Long tag with a lot text</span>
    </div>
  </div>
</Canvas>

Tips: You can add tooltip for long tag content to display its content inside
