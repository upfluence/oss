# Flexbox

The Flexbox system aims to provide a more efficient way of arranging, aligning and distributing space between the elements
of a container. The goal of the flexible layout is to give the container the ability to change the width and height of its
elements to best fill the available space.

Keep in mind that the alignment of the axes are switched based on the flex-direction:

- flex-direction: row : &larr; Main Axis &rarr; / &uarr; Cross &darr;
- flex-direction: column : &larr; Cross Axis &rarr; / &uarr; Main &darr;

## Flexbox properties for the Children (flex items)

`fx-1` class equivalent of `{flex: 1}`

## flex-basis
Usage: fx-basis-pc-XX 
where: XX is a percentage

flex-basis in percentage classes are pre-generated from 0 to 100 with 5% steps.
fx-basis-pc-0, fx-basis-pc-5, fx-basis-pc-10... up until 100
For 1/3 & 2/3 of space usage, you'll also find fx-basis-pc-33 & fx-basis-pc-66

## flex-gap
Usage: fx-gap-pc-XX 
where: XX is a percentage
or
Usage: fx-gap-px-XX 
where: XX is a size in pixels

flex-basis in percentage classes are pre-generated from 0 to 100 with 5% steps.
fx-gap-pc-0, fx-gap-pc-5, fx-gap-pc-10... up until 100
fx-gap-px-0, fx-gap-px-5, fx-gap-px-10... up until 100

## flex-wrap
Usage: fx-wrap

## Flexbox properties for the Parent (flex container)

The following classes are to be put in the parent container

### Flex direction definition

`fx-row` class equivalent of `{display: flex; flex-direction: row;}`

`fx-col` class equivalent of `{display: flex; flex-direction: column;}`

### Main axis alignment

`fx-malign-start` class equivalent of `{justify-content: flex-start;}`

`fx-malign-center` class equivalent of `{justify-content: center;}`

`fx-malign-end` class equivalent of `{justify-content: flex-end;}`

`fx-malign-space-between` class equivalent of `{justify-content: space-between;}`

`fx-malign-space-around` class equivalent of `{justify-content: space-around;}`

`fx-malign-space-evenly` class equivalent of `{justify-content: space-evenly;}`

### Cross axis alignment

`fx-xalign-start` class equivalent of `{align-items: flex-start;}`

`fx-xalign-center` class equivalent of `{align-items: center;}`

`fx-xalign-end` class equivalent of `{align-items: flex-end;}`
